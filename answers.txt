Q1: Why might we consider using Git?

Answer: Git is the most popular Version Control System. It allows multiple people to manage changes
to a source code file and allows users to go back to previous versions of the code.

Answer: Data sience could be applied in multiple fields. Practically everything has and uses data,
and even if a company or feild is not explicitly using that data, it can be used to find trends
and make improvemets to data and workflow. Data sience could be used at a bakery to find the most
